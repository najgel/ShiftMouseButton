ShiftMouseButton

A small Windows tray utility to swap the primary mouse button (left/right) with a hotkey.

Download

Get the latest release from GitHub:

View all releases

Choose one of:

  • MSI – Recommended. Installs to Program Files and adds an uninstaller.
  • Portable ZIP – Extract and run. Use "Run at Startup" from the tray if desired.

Requires Windows and .NET 8.0 Desktop Runtime (if not using a self-contained build).

Features

  • Global hotkey (default Ctrl+Alt+M) to swap mouse buttons – configurable
  • System tray icon – left-click to swap, right-click for menu
  • Run at Windows startup option
