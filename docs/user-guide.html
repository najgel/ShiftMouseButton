ShiftMouseButton – User Guide

Installation

MSI (recommended)

Download ShiftMouseButton-x.x.x.msi from the releases page, run the installer, and follow the prompts. You can optionally check "Run ShiftMouseButton at Windows startup and launch now" on the final screen.

Portable

Download the ZIP, extract it, and run ShiftMouseButton.exe. If you want it to start with Windows, use the tray menu → "Run at Startup".

Usage

Swap mouse buttons

  • Press your configured hotkey (default Ctrl+Alt+M)
  • Or left-click the system tray icon
  • Or right-click the tray icon → "Swap Mouse Buttons (...)"

Change the hotkey

Right-click the tray icon → "Hotkey...". Pick modifiers (Ctrl, Alt, Shift, Win) and a key. Settings are saved in %AppData%\ShiftMouseButton\settings.json.

You can also override on the command line: ShiftMouseButton.exe --hotkey "Ctrl+Alt+M" or --hotkey=Ctrl+Alt+N.

Run at startup

Right-click the tray icon → "Run at Startup". A checkmark indicates it is enabled.

Exit

Right-click the tray icon → "Exit".

Requirements

  • Windows
  • .NET 8.0 Desktop Runtime (unless using a build that includes the runtime)

Troubleshooting

If you see "Failed to register hotkey", another application (or another instance) may be using that hotkey. Choose a different combination or close the other app.
